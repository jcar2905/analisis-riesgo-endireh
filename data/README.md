# Datos — ENDIREH 2021

Los datos no se incluyen en este repositorio por razones de privacidad 
y tamaño del archivo.

## Cómo obtenerlos

1. Visita: https://www.inegi.org.mx/programas/endireh/2021/
2. Descarga los microdatos en formato .dta
3. Coloca los siguientes archivos en esta carpeta:
   - `TB_SEC_IVaVD.dta`
   - `TSDem.dta`
